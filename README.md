## How to run:

Download the repository as a zip. Unzip and navigate to the extracted folder.

Compile all files:

```bash
    javac RSAKeyGenerator.java
    javac Sender.java
    javac Receiver.java
```

Then run the compiled files:

```bash
    java RSAKeyGenerator
    java Sender
    // Transmitted_Data.txt is created
    java Receiver
```
